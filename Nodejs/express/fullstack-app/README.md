# Short URL application

## Description
This is a simple application that allows you to shorten the URL.

## Installation
1. Install the dependencies
   `npm install`
2. Run the application
   `npm start` or `node index.js` or `nodemon index.js`

## Dependencies
- express
- mongoose
- cors
- dotenv
- sweetalert2
- bootstrap
- body-parser
